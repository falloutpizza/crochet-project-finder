# what will you crochet today?

a website where you can easily find new crochet ideas, instead of having to scroll through youtube or pinterest for hours. easily use the filter functionality to find the exact project you want to make today, from clothing to room decor, from projects requiring mutliple skeins of yarn to just half a skein.

currently deployed at: https://falloutpizza.github.io/crochet-project-finder/
